# Front-End Gerenciamento de Cursos

Interface web para consumir a API REST de gerenciamento de cursos, desenvolvida como complemento da atividade de Spring Boot.

## API

Este front-end consome a API disponível em:
> [API-REST-SpringBoot](https://github.com/lopreti/API-REST-SpringBoot) — rode localmente em `http://localhost:8080`

## Tecnologias

| Tecnologia | Descrição |
|---|---|
| React | Biblioteca |
| TypeScript | Linguagem principal |
| CSS | Estilização |
| HTML | Estrutura das páginas |
| JavaScript | Scripts auxiliares |

## Como rodar localmente

### Pré-requisitos
- Node.js instalado
- API back-end rodando em `http://localhost:8080`

### Passo a passo

1. Clone o repositório:
```bash
git clone https://github.com/lopreti/Front-End-API-REST-SpringBoot.git
```

2. Entre na pasta do projeto:
```bash
cd Front-End-API-REST-SpringBoot/frontend
```

3. Instale as dependências:
```bash
npm install
```

4. Rode o projeto:
```bash
npm run dev
```

> ⚠️ Certifique-se de que a API está rodando antes de abrir o front-end.
